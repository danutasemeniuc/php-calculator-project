# PHP Calculator Project

Proiect simplu PHP pentru demonstrarea CI/CD cu Jenkins.

## Instalare
```bash
composer install
```

## Rulare Teste
```bash
./vendor/bin/phpunit
```

## Funcționalități

### Calculator
- Adunare, scădere, înmulțire, împărțire
- Ridicare la putere
- Calcul medie

### StringHelper
- Inversare string
- Numărare cuvinte
- Verificare palindrom
- Conversie upper/lower case
